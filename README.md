# MidMarksSystem 🎓📊

A Java-based web application to manage and track student mid-exam marks efficiently for faculty and HODs.

---

## 📌 Project Features

- 🧑‍🏫 **Faculty Login** – Secure login by department, view & update student mid marks by subject.
- 🧑‍💼 **HOD Login** – Secure login, view mid marks of all students, generate internal averages, and manage:
  - Add/Remove Faculty
  - Add/Remove Subjects
  - Add/Promote Batches
  - Update Faculty & Student Details

---

## 🛠️ Technologies Used

- **Frontend**: JSP, HTML, CSS
- **Backend**: Java Servlets
- **Database**: MySQL (via XAMPP)
- **Web Server**: Apache Tomcat
- **IDE**: Eclipse

## 🚀 Running the Project Locally

1. Open **XAMPP**, start **Apache** and **MySQL**.
2. Create database: `midmarks_db` and import tables.
3. Open **Eclipse** and import the project folder.
4. Make sure Apache Tomcat is configured.
5. Right-click project → Run on Server.
6. Access from browser:  
   `http://localhost:8080/MidMarksSystem`
